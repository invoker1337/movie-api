Movie API Documentation

Title: Achievement 2 Project: myFlix

Objective: To build the server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movie.

Endpoints

REQUEST URL HTTP METHOD REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of ALL movies to the user /movies GET NONE A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:movieTitle GET NONE A JSON object holding data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/:genreName GET NONE A JSON object holding data about a moviegenre
Return data about a director (bio, birth year, death year) by name /directors/:directorName GET NONE A JSON object holding data about a director
Allow new users to register /user/register POST NONE a text message indicating wether the user was succesfully registered
Allow users to update their user info (username) /user/:username PUT NONE a text message indicating wether the userinfo was succesfully updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) user/favorites/:movieTitle PUT JSON object holding data about the movie to add a text message indicating wether the movie has been added to the list of favorites
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) user/favorites/:movieTitle DELETE NONE a text message indicating wether the movie was removed from their list of favorites
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) user/:username DELETE NONE a text message indicating wether the user was succesfully deregistered
